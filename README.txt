This program is developed to register two point sets with similarity transformation.

The source code and the installing package are both provided in this directory.

The source code is implemented in C++ programming language, which employs the Visualization Toolkit (www.vtk.org) for the 3D visualization and QT for the UI designing. You may use CMake (www.cmake.org) to configure the program under any platform (Linux, Windows or OS). The source code is in the sub-directory of 3PCHM-src.

The installation package can only be used under the windows system (win7 is recommended), and the package is in the sub-directory of 3PCHM-installer.
